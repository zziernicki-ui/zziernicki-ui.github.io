Case study 01 · Applied AI

Multimodal Image Captioning Web App

Solo developer · Python · BLIP · Gradio

A browser tool that generates natural-language captions for any uploaded image, built around Hugging Face's BLIP vision-language transformer and served through Gradio.

  • Built the full pipeline: image pre-processing → BLIP inference → token decoding → web UI.
  • Progressed iteratively from a CLI proof-of-concept to a packaged function to a deployed interface.
  • Verified the model generalizes beyond its test image (certificate and personal photo).

Case study 02 · Applied AI

Conversational AI Chatbot

Solo developer · Flask · Hugging Face · JavaScript

A full-stack chatbot: a browser chat page talks to a Flask API that hosts a Hugging Face language model and answers in a continuous, multi-turn conversation.

  • Wired three layers together — JS front end, Flask /chatbot API, and BlenderBot inference loaded once at startup.
  • Solved the real integration problems: CORS across origins and conversation-state across turns.
  • Evaluated two models (deployed BlenderBot, compared SmolLM2); verified at CLI, API, and browser.

I started in enterprise IT — imaging fleets, administering Active Directory, leading a national WLAN deployment — and moved toward building AI applications that actually ship.
