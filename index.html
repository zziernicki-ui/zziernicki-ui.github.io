Case study 03 · Applied AI

Zack's IT Help Desk — A Full-Stack RAG System

Solo developer · FastAPI · Chroma · Claude API

A retrieval-augmented troubleshooting assistant, live on Azure: a React chat page talks to a FastAPI back end that searches a knowledge base semantically and has Claude write grounded, step-by-step answers that cite their sources.

  • Built the RAG pipeline in phases: keyword proof-of-concept → Chroma semantic search → multi-format ingestion with chunking and source citations.
  • Evaluated vendors under real constraints — pivoted generation from IBM watsonx to the Claude API without touching the surrounding layers.
  • Debugged the Azure App Service deployment from its logs (worker configuration, ~4 GB of unused dependencies) and verified the live system adversarially.

Case study 04 · Applied AI

The Azure Migration — Cloud-Native, Zero-Secret RAG

Solo developer · Azure AI Search · Azure OpenAI · GitHub Actions

The complete re-platforming of the help desk onto Azure-native managed services: continuous deployment, persistent hybrid retrieval, Azure-hosted generation, production telemetry — and zero standing secrets.

  • Replaced the custom loader, chunker, and vector store with Azure AI Search integrated vectorization — three modules deleted, the app got smaller while the system does more.
  • Eliminated every API key with a system-assigned Managed Identity, RBAC, and Key Vault — diagnosing a persistent 401 by decoding the app's identity token over SSH.
  • Absorbed two forced model pivots (a zero Claude quota, a retired gpt-4o-mini) and shipped on gpt-5-mini, with the generator's structure unchanged.
